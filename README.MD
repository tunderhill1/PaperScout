# PaperScout

PaperScout is an AI-powered research paper discovery and recommendation system.
It combines semantic search (via embeddings), vector similarity (pgvector), and
optional AI summarisation to help users find relevant academic papers.

## Tech Stack
- FastAPI
- Python 3.12+
- PostgreSQL + pgvector
- Docker
- GCP (Cloud Run + Cloud SQL)
